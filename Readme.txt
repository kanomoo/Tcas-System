==============================================
          TCAS Admission Management System
==============================================

ระบบจัดการการสมัครสอบ TCAS สำหรับนักเรียนที่ต้องการสมัครเข้ามหาวิทยาลัย

----------------------------------------------
  การติดตั้ง (Installation)
----------------------------------------------
1. ต้องติดตั้ง Python 3.10 ขึ้นไป
2. ติดตั้งแพ็กเกจที่จำเป็นด้วยคำสั่ง:

   pip install wcwidth

----------------------------------------------
  ความสามารถของระบบ (Features)
----------------------------------------------
1. เมนูนักเรียน (Student Menu)
   - ลงทะเบียนข้อมูลนักเรียน
   - จัดการข้อมูลการสมัคร

2. ข้อมูลหลักสูตร (Course Information)
   - ค้นหาข้อมูลคณะและหลักสูตร
   - ดูรายละเอียดการรับสมัคร

3. รายงานสถิติ (Statistics Report)
   - รายงานจำนวนผู้สมัคร
   - รายงานค่าสมัครสอบ

----------------------------------------------
  วิธีการใช้งาน (Usage)
----------------------------------------------
1. รันโปรแกรมหลักโดยใช้คำสั่ง:

   python main.py

2. เลือกเมนูการใช้งาน:
   [1] เมนูนักเรียน
   [2] ข้อมูลหลักสูตร
   [3] รายงานสถิติ
   [4] ออกจากโปรแกรม
